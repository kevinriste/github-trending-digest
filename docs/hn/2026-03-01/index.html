No 08  ·  Blueprint  ·  Build Log

Show HN: Xmloxide – an agent-made Rust replacement for libxml2

github.com  ·  64 pts  ·  63 comments  ·  discussion

Analysis

Developer Jon Wiggins has introduced "xmloxide," a memory-safe, pure Rust reimplementation of the long-standing libxml2 library. Following the official end of maintenance for libxml2 in late 2025 due to unresolved security issues, this new package serves as a drop-in replacement that includes a C API for compatibility with existing software. The project achieved full conformance with the W3C XML test suite and offers performance levels comparable to or better than the original C library.
