Technical Analysis

`Enject` (formerly known as `enveil`) is a Rust-based utility designed to protect `.env` file secrets from being exposed to AI coding agents and LLM tools. Instead of storing plaintext credentials on disk, it uses an encrypted local store to hold secret values, while the `.env` file contains only symbolic `en://` references. When a user runs a command, the tool prompts for a master password to decrypt the secrets in memory and injects them directly into the subprocess at launch, ensuring plaintext sensitive data is never written to the filesystem.
