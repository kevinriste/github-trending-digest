Developer Narek Maloyan has successfully ported the Manim engine to TypeScript, enabling complex mathematical visualizations directly in the browser. Read →Share →
[ Analysis + ]

Technical Analysis

Narek Maloyan has released Manim-Web, a TypeScript implementation of 3Blue1Brown’s popular Manim math animation engine that runs entirely in the browser. By leveraging the Canvas API, WebGL via Three.js, and KaTeX for mathematical rendering, the project eliminates the need for complex local dependencies like Python, FFmpeg, and LaTeX distributions. It maintains an API structure nearly identical to the original Python version and includes a conversion tool to help developers migrate existing scripts to the web environment.

Designing the Smallest Transformer for Arithmetic Tasks

github.com  ·  247 pts  ·  99 comments

The AdderBoard project challenges the community to build the most parameter-efficient transformer capable of addition. Participants are iterating on architectural constraints to discover how minimal models handle carry propagation and alignment.

Read →Share →
[ Analysis + ]

Technical Analysis

AdderBoard is a collaborative research project and leaderboard dedicated to finding the smallest possible autoregressive transformer capable of adding two 10-digit numbers with at least 99% accuracy. Participants submit models under two categories: hand-coded architectures that demonstrate theoretical feasibility and trained models that use generic learning algorithms to achieve the result. The project tracks parameter counts and architectural innovations, such as weight tying, low-rank factorization, and circular arc embeddings, to identify the minimal structures necessary for digit alignment and carry propagation.
