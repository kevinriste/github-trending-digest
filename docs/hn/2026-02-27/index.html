No 08  ·  Academic  ·  Security Research

AirSnitch: Demystifying and breaking client isolation in Wi-Fi networks [pdf]

ndss-symposium.org  ·  412 pts  ·  193 comments  ·  discussion

Analysis

The research paper "AirSnitch" provides a comprehensive security analysis of client isolation in Wi-Fi networks, demonstrating that this widely deployed defensive mechanism is frequently ineffective across home and enterprise environments. The authors identify that vendors often implement isolation inconsistently, leaving vulnerabilities in the Wi-Fi encryption, packet switching, and routing layers. By exploiting these flaws—such as abusing shared Group Temporal Keys (GTKs), leveraging gateway-level routing, and performing MAC address spoofing—an attacker can bypass existing protections to conduct full machine-in-the-middle attacks.

No 10  ·  Blueprint  ·  Show HN

Parakeet.cpp – Parakeet ASR inference in pure C++ with Metal GPU acceleration

github.com  ·  114 pts  ·  30 comments  ·  discussion

Analysis

Parakeet.cpp is a high-performance, pure C++ implementation for running NVIDIA’s Parakeet automatic speech recognition (ASR) models. By utilizing a lightweight tensor library called axiom, the project achieves Metal GPU acceleration on Apple Silicon without relying on heavy dependencies like PyTorch or ONNX Runtime. It supports a variety of offline and streaming model types, offering features such as word-level timestamps, speaker diarization, and phrase boosting through an efficient, portable codebase.
