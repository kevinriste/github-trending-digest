No 03  ·  Blueprint  ·  Maker Projects

Show HN: Llama 3.1 70B on a single RTX 3090 via NVMe-to-GPU bypassing the CPU

github.com  ·  393 pts  ·  101 comments  ·  discussion

Analysis

NTransformer is a high-efficiency C++/CUDA inference engine designed to run massive transformer models, such as Llama 3.1 70B, on consumer hardware like a single RTX 3090. The project achieves this by bypassing the CPU through a three-tier caching system that utilizes VRAM, system RAM, and direct NVMe-to-GPU streaming. It features an advanced pipeline that allows for asynchronous data transfers and layer skipping to maintain functionality despite the significant memory constraints of consumer-grade GPUs.
