Technical Analysis

A researcher has discovered that the macOS version of NetEase’s MuMu Player Pro silently executes seventeen reconnaissance commands every thirty minutes while the software is running. These background routines harvest a wide range of sensitive system data, including local network device enumerations, full lists of running processes with command-line arguments, installed application metadata, and host file contents. This information is linked to the user's hardware serial number and transmitted to the Chinese analytics platform SensorsData, despite these activities being absent from the application's privacy policy.

Technical Analysis

Pi for Excel is an open-source AI sidebar add-in that integrates large language models directly into Microsoft Excel spreadsheets. Built using the Pi agent framework, the tool allows users to connect their own API keys from providers like Anthropic, OpenAI, Google Gemini, or GitHub Copilot to interact with workbook data. It provides 16 core tools for tasks such as formula explanation, workbook structure navigation, and cell formatting, while maintaining session history and automatic recovery checkpoints for safety.
