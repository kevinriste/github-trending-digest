No 04  ·  Blueprint  ·  Engineering Build

BarraCUDA Open-source CUDA compiler targeting AMD GPUs

github.com  ·  459 pts  ·  201 comments  ·  discussion

Analysis

BarraCUDA is an open-source C99 compiler that translates CUDA C++ files into machine code for AMD GPUs, NVIDIA PTX, or Tenstorrent Tensix architectures. Written from scratch without LLVM or external dependencies, the project aims to bypass the vendor-locked ecosystems typically associated with GPU programming. It includes its own instruction encoder, a divergence-aware register allocator, and a lightweight runtime for executing compiled kernels on hardware.

No 07  ·  Terminal  ·  Systems Programming

Minimal x86 Kernel Zig

github.com  ·  155 pts  ·  64 comments  ·  discussion

Analysis

This project introduces a minimal, bare-metal x86 kernel written entirely in the Zig programming language, eliminating the need for traditional assembly files. By targeting `x86-freestanding-none` and utilizing the Multiboot 1 protocol, the kernel demonstrates how to interact directly with hardware, such as the VGA text buffer, using Zig's memory-mapped I/O capabilities. It is designed to be cross-compiled from diverse host environments, including Apple Silicon, and tested via QEMU or GRUB-enabled ISOs.
