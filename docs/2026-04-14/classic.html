Kronos is a specialized, open-source foundation model family designed specifically to interpret the "language" of financial markets through candlestick (K-line) sequences. Unlike general-purpose time-series models, it employs a sophisticated two-stage framework that first tokenizes multi-dimensional OHLCV data into hierarchical discrete representations. The architecture supports a range of model sizes, from 4.1M to over 499M parameters, providing flexibility for diverse computational needs. Users can easily implement these models for quantitative forecasting tasks via a streamlined Python API that automates normalization and inference.
